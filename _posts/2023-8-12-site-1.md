## 준비물
python3, Django, PowerShell

## 1단계
PowerShell을 켜서 다음 명령어를 실행시킨다.

```bat
cd \
mkdir venvs
cd .\venvs\
python -m venv mysite
cd .\mysite\
cd .\Scripts\
.\activate
```
성공한 화면
![image](/assets/images/site/1/mysite_activate.png)

## 2단계
```bat
cd \
mkdir projects
cd .\projects\
mkdir mysite
cd .\mysite\
```
여기서 장고를 설치하지 않았다면
```bat
python -m pip install --upgrade pip

```
