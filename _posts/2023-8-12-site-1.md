## 준비물
python3, Django

## 1단계
```bash
cd \
mkdir venvs
cd .\venvs\
python -m venv mysite
cd .\mysite\
cd .\Scripts\
.\activate
```
![img](assets\images\site\1\mysite_activate.png)
성공한 화면

## 2단계
```bash
cd \
mkdir projects
cd .\projects\
mkdir mysite
cd .\mysite\
```
여기서 장고를 설치하지 않았다면
```bash
pip install django
```

```bash
django-admin startproject config .
python .\manage.py runserver
```

![img](assets\images\site\1\django_start.png)
