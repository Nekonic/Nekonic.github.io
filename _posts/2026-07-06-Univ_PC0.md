실습실의 근로를 맡고 있다. 교수님의 지시사항으로, 외국인 대학생들을 위한 PC 관리를 해야한다.  
PC 배경화면, 프로그램 설치 및 동일한 환경 구성 등등 여러가지 있는데, 그걸 어떻게 해결할 수 있을까?  

## AD DS(Active Directory Domain Services)
Windows 네트워크 환경에서 사용자, 컴퓨터, 그룹, 정책 등을 중앙에서 관리하는 디렉터리 서비스

## WDS(Windows Deployment Services)
Windows 설치 파일을 네트워크를 통해 제공하는 서비스  
윈도우 설치를 위한 서버

## MDT(Microsoft Deployment Toolkit)
Windows를 자동으로 설치하고 설정하는 배포 도구

## PXE (Preboot Execution Environment)
네트워크를 통해 부팅하는 기술

## 실습환경 준비

### Vmware

| | 역할 | OS | 사양 |
| - | - | - | - |
| win2022 | AD DS, DHCP, WDS, MDT | Windows Server 2022 | CPU 2, RAM 4GB Disk 50GB |
| RefPC | 레퍼런스 이미지 빌드 및 캡쳐 | Windows 11 Pro | CPU 2, RAM 4GB Disk 50GB |
| ClientPC | 캡처된 이미지 배포 테스트 | 빈 디스크 | CPU 2, RAM 4GB Disk 50GB |
